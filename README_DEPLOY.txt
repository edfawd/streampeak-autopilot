FREE HOSTING WITH GITHUB PAGES
1. Create or sign in to a GitHub account.
2. Create a PUBLIC repository called:
   streampeak-autopilot
3. Upload every file from this folder to the repository root.
4. Open repository Settings -> Pages.
5. Under Build and deployment choose:
   Deploy from a branch
6. Select branch:
   main
   and folder:
   / (root)
7. Save and wait a few minutes.

Your URLs will normally look like:
https://YOUR-GITHUB-NAME.github.io/streampeak-autopilot/
https://YOUR-GITHUB-NAME.github.io/streampeak-autopilot/privacy.html
https://YOUR-GITHUB-NAME.github.io/streampeak-autopilot/terms.html
